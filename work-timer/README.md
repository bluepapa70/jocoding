# 안랩 업무 시간 타이머

업무 시간을 시각적으로 표시하는 플립 시계 기반 웹 타이머입니다.

## 기능

- **플립 시계** — 현재 시각을 실제 플립 시계 애니메이션으로 표시
- **세션 자동 전환** — 오전 근무 / 점심 / 오후 근무 / 퇴근 후 자동 구분
- **진행률 바** — 현재 세션의 경과율 실시간 표시
- **주간 달력** — 이번 주 일~토 달력, 오늘 날짜 강조
- **일과 타임라인** — 하루 근무 흐름 시각화
- **스트레칭 알림** — 매시 50분 전체화면 알림 (60초 카운트다운)
- **화면보호기 모드** — F키 또는 버튼으로 전체화면 대형 시계 전환

## 근무 시간표

| 구분 | 시간 |
|------|------|
| 오전 근무 | 09:00 ~ 11:30 |
| 점심 시간 | 11:30 ~ 12:30 |
| 오후 근무 | 12:30 ~ 18:00 |

## 실행 방법

### 방법 1 — 브라우저에서 직접 열기

```bash
# macOS
open work-timer.html

# Windows
start work-timer.html

# Linux
xdg-open work-timer.html
```

### 방법 2 — 로컬 서버로 실행 (권장)

```bash
# Python 3
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080/work-timer.html 접속

# Node.js (npx)
npx serve .
```

## 화면보호기 사용법

| 동작 | 방법 |
|------|------|
| 진입 | 하단 **"화면보호기 시작"** 클릭 또는 키보드 **F** |
| 해제 | 화면 클릭 또는 아무 키 |

## Claude Code에서 수정하기

이 프로젝트는 단일 HTML 파일(`work-timer.html`)로 구성되어 있습니다.
Claude Code로 수정 요청 예시:

```
claude "스트레칭 알림 시간을 매시 45분으로 바꿔줘"
claude "퇴근 시간을 18시 30분으로 변경해줘"
claude "플립 시계 카드 크기를 키워줘"
```
